📌 Image Steganography using Python

🔍 Overview

A Python-based application to securely hide and extract secret messages within images using steganography techniques.

🚀 Features

* Hide text inside images
* Extract hidden messages
* Maintains image quality
* Simple GUI interface

🛠️ Tech Stack

Python, OpenCV, NumPy, Tkinter

▶️ How to Run

pip install opencv-python numpy
python main.py

📈 Learning

Gained hands-on experience in image processing, data hiding, and Python-based application development.
